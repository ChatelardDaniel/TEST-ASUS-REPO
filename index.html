Cours introduction aux lignes de commande.

Les commandes de base.

Commande Windows

dir, permet de lister les fichiers et dossiers qui ne sont pas cachés.

dir /a, avec l'option /a on affichera les fichiers et dossiers cachés.

dir /a nomDuDossier, affichera le contenu du dossier demandé.

., point est le dossier courant.

.., point point est le dossier parent.

ipconfig, voir l'adresse IP d'une machine Windows.

Commande Linux

ls, permet de lister les fichiers et dossiers qui ne sont pas cachés.

ls -a, avec l'option -a on affichera les fichiers et dossiers cachés.

ls -a nomDuDossier, affichera le contenu du dossier demandé.

., point est le dossier courant.

.., point point est le dossier parent.

ip a, voir l'adresse IP d'une machine Linux.

ipconfig, voir l'adresse IP d'une machine macOS.

Manipuler des dossiers et des fichiers.

Commande Windows

mkdir nomDuDossier, permet de créer un dossier.

echo "bonjour", affichera du texte dans la console.

echo "bonjour" > windows.txt, créera un fichier windows.txt avec comme contenu bonjour.

move bonjour.txt .., déplacer le fichier bonjour.txt vers le dossier parent.

move subfolder windows, déplacer le dossier subfolder dans le dossier windows.

move texte.txt texte1.txt, Renomme le fichier texte.txt en texte1.txt

del texte1.txt, supprime le fichier texte1.txt.

rmdir repertoire, supprime le répertoire "repetoire" du répertoire courant.

Commande Linux

mkdir nomDuDossier, permet de créer un dossier.

touch linux.txt, créer un fichier nommé linux.txt.

echo "bonjour" > linux.txt, créera un fichier linux.txt avec comme contenu bonjour.

mv bonjour.txt .., déplacer le fichier bonjour.txt vers le dossier parent.

mv subfolder windows, déplacer le dossier subfolder dans le dossier windows.

mv texte.txt texte1.txt, Renomme le fichier texte.txt en texte1.txt

rm texte1.txt, supprime le fichier texte1.txt. idem sur macOS.

rmdir repertoire, supprime le répertoire "repetoire" du répertoire courant.

rm -r repertoire, supprime le répertoire "repertoire" et son contenu du répertoire courant. -r pour récursive, cela veut dire que la commande va supprimer tous les fichiers, dossiers et sous-dossiers du répertoire indiqué.
