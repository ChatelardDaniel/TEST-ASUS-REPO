Cours introduction aux lignes de commande.

Commande Windows

dir, permet de lister les fichiers et dossiers qui ne sont pas cachés.

dir /a, avec l'option /a on affichera les fichiers et dossiers cachés.

dir /a nomDuDossier, affichera le contenu du dossier demandé.

., point est le dossier courant.

.., point point est le dossier parent.

Commande Linux

ls, permet de lister les fichiers et dossiers qui ne sont pas cachés.

ls -a, avec l'option -a on affichera les fichiers et dossiers cachés.

ls -a nomDuDossier, affichera le contenu du dossier demandé.

., point est le dossier courant.

.., point point est le dossier parent.
